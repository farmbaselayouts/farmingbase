# FWCB Farming Base Layout

A professional and dynamic web platform designed for the Clash of Clans community to easily find, share, and copy optimized farming base layouts. The system categorizes layouts by Town Hall levels and features a fully integrated Admin Panel for seamless content management.

---

### Key Features
* **Dynamic Base Directory:** Automatically organizes and sorts base layouts by Town Hall level (Highest to Lowest).
* **Secure Admin Dashboard:** A robust control panel allowing authorized users to add, edit, and delete base layouts, as well as update community rules in real-time.
* **One-Click Copy:** Integrated links allow users to directly copy the base layouts into their Clash of Clans game.
* **Cloud Database:** Powered by Google Firebase for instant, real-time data syncing without requiring manual codebase updates.
* **Responsive UI:** A premium dark-themed interface built for seamless performance across desktop, tablet, and mobile devices.

### Technology Stack
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6)
* **Backend & Database:** Google Firebase (Cloud Firestore & Authentication)
* **Deployment:** GitHub Pages

---

### Setup Instructions
1. **Clone the Repository:** Download or clone this repository to your local system.
2. **Configure Firebase:** Update the Firebase configuration block in `index.html`, `rules.html`, and `admin.html` with your specific Firebase project credentials.
3. **Assets:** Ensure the `logo.png` file is placed in the root directory for site branding.
4. **Deploy:** Push the updated files to your GitHub repository and enable GitHub Pages from the repository settings to launch the website.
